# Bearsmacs

**How to Start:**

    git clone https://github.com/Abergard/bearsmacs.git ~/.emacs.d
    cd ~/.emacs.d
    mkdir emacs-autosaves
    mkdir tmp

**User configuration file:**

  To make any configuration changes use: ~/.bearsmacs.el

**Hierarchy of files:**
  - init.el
  - private/
    - bears-bind.el
    - bears-packages.el
    - bears-style.el
    - packages/
      - bears-package-name.el
      - ...
    - themes/
      - bears-theme-name.el
      - ...
