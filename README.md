# Bearsmacs

**How to Start:**

    git clone https://github.com/Abergard/bearsmacs.git ~/.emacs.d

**User configuration file:**

  To make any configuration changes use: ~/.bearsmacs.el

**Hierarchy of files:**
  - init.el
  - private/
    - bears-bind.el
    - bears-configuration.el
    - bears-defaults.el
    - bears-functions.el
    - bears-packages.el
    - bears-style.el
    - bears-user-file.el
    - bears-variables.el
    - packages/
      - bears-(package-name).el
      - ...
    - themes/
      - bears-(theme-name).el
      - ...
